# UX Writing Skill for Claude

A professional Claude Skill for reviewing and optimizing interface copy, including buttons, labels, error messages, CTAs, tooltips, and notifications.

## What's Included

- **10 Core Principles**: Concise, user-centered guidelines for effective UX writing
- **Tone Framework**: Guidance for adapting copy based on context and user emotion
- **UX Writing Checklist**: Quick reference for optimizing interface copy

## Installation

1. Download `ux-writing.skill` from this repository
2. Go to [Claude.ai](https://claude.ai)
3. Navigate to Settings â†’ Skills

## Usage

Once installed, this skill will automatically trigger when you:
- Review or optimize UI text
- Analyze interface copy in screenshots
- Improve button labels, error messages, or CTAs
- Audit microcopy for digital products

## Key Features

### Core Principles
- Be Concise
- Speak the User's Language
- Be Consistent
- Use Specific, Action-Oriented CTAs
- Lead with Important Information
- Write in Active Voice
- Be Specific About What Happens
- Adopt a Conversational Tone
- Never Blame the User
- Design for Accessibility

### Tone Frameworks
- Four tone dimensions (Humor, Formality, Respectfulness, Enthusiasm)
- Contextual tone matrix for different situations
- User state adaptation guidelines
- Brand voice archetypes

## Example Use Cases
